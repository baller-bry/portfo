Offering Count

The program begins by checking if the Excel file already exists, if so, it will simply open the existing file. If the file has not yet been created, meaning the program has not been run for that day, then it will copy the template file onto the created target file using the copyfile function from the shutil module. After the file has been created, the program checks if the machine's operating system is Windows. If it is, it will open the Windows file, if it's not, it will assume the machine is macOS and will open the file accordingly.

Note: Because the program is run on macOS in the video below, the Windows code segments of the source code are commented out because macOS does not recognize the modules/functions.

See below for the source code and the video demonstrating how the program behaves when run using a .command shell script.
