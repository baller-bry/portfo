Encrypter/Decrypter

Using the fernet module from the cryptography package, the first program displays a password encrypter dialog that asks the user to input a password. After the user inputs a password, a key (ciphertext as type bytes) is generated and written to a key.txt file decoded as a string. A Fernet object is created with the key and encrypts the password encoded as bytes. The encrypted password is written to an encryption.txt file decoded as a string.

The second program reads in the key.txt file, re-encodes the string key ciphertext into bytes, and generates a Fernet object with the key. The encryption.txt file is read in and the string encryption ciphertext also gets re-encoded into bytes. The decrypted password is retrieved by decrypting the encoded encryption which is finally decoded back into a string. The program displays a password decrypter dialog and reveals the decrypted password.

See below for the source code (two programs) and a video demonstrating each program's behavior when run. The programs are run on Windows using shell scripts (.sh) that were created with Git Bash.
